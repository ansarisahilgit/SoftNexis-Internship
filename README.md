# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/ansarisahilgit/SoftNexis-Internship.git
```

---

## Backend Setup

```bash
cd backend

npm install

npm run dev
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm start
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **backend** directory.

```env
PORT=3000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
